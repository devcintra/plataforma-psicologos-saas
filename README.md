# Plataforma de Conexão entre Psicólogos e Pacientes

## Descrição
Este projeto tem como objetivo conectar psicólogos iniciantes a pacientes que buscam atendimento psicológico acessível.

Psicólogos recém-formados enfrentam dificuldades para conseguir seus primeiros pacientes, enquanto muitas pessoas encontram barreiras no acesso à saúde mental.

A plataforma resolve esse problema permitindo agendamento de consultas, visualização de perfis e avaliações.

## Modelo de Negócio
SaaS (Software as a Service):
- Gratuito para pacientes
- Assinatura para psicólogos

## Público-Alvo
- Psicólogos iniciantes (0–3 anos de formação)
- Pessoas que buscam atendimento psicológico acessível

## Impacto Social
Facilitar o acesso à saúde mental e ajudar profissionais no início da carreira.
